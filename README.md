MamaPutOnline Landing Page

Files:
- index.html: complete landing page code with comments and visible prompts used per section.
- assets/: downloaded and resized food images used in the product cards.

How to use:
1. Open index.html in a browser.
2. Submit the index.html file and assets folder together, or submit the zipped folder.
3. The search bar and category filters work with small built-in JavaScript.
4. The colours are blue, white and purple as requested.
5. The menu and footer are sticky.

Note:
The images are from Wikimedia Commons and were resized locally for web display. Replace them with original restaurant photographs before using this page commercially.
