Prompts used per section

These are the vibe-engineering prompts used to shape the page structure, colour direction, layout and interaction.

Visible assignment evidence

Overall UI / UX Prompt

Design a clean landing page for MamaPutOnline, an online takeaway Nigerian cooked meal restaurant. Use a blue, white and purple colour system, strong visual hierarchy, readable product cards, and a simple layout that feels modern but still Nigerian-food focused.

Menu Prompt

Create a sticky top menu with the MamaPutOnline logo on the top-left, navigation links for Products, Services, Customer Service and Contact Us, and a search input on the top-right. Use a dark blue/purple gradient and clear hover states.

Hero Prompt

Create a hero section that occupies at least 30% of the vertical page height and 100% of the page width. Split it into two columns: one column should contain bold persuasive text and call-to-action buttons; the other should show a steaming Nigerian soup image with correct aspect ratio and sharp display.

Main/Product Card Prompt

Create a main section with a sticky side panel on the left and product cards on the right. The side panel should link to food categories and specific dishes. The product grid should have at least three rows and four cards per row on desktop. Each card should show image, product name, quantity, price in Naira and discount.

Image Sourcing Prompt

Find online images of Nigerian prepared foods such as egusi, ewedu, ora/oha, edikaikong, okazi, ogbono, okro, jollof rice, fried rice, porridge yam, beans, plantain, abacha, nkwobi, fried chicken and kpomo. Download them locally, resize them for web use, and use them inside product cards.

Footer Prompt

Create a sticky footer with social media links, basic contact identity and a compact layout. Keep it in the same blue/purple brand style and make sure it stays visible at the bottom of the screen.
